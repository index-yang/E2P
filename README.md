## :wrench: Dependencies and Installation
- CUDA = 11.7 (Only tested on this version)
- Both Windows and Linux can run this code
- Other required packages in requirements.txt
#### Step1. Create conda environment
```
# Clone this repo.
git clone https://github.com/index-yang/E2P
cd E2P

# Create conda env.
conda create -n E2P python=3.9
conda activate E2P
```
#### Step2. Install dependencies
```
# Install torch
pip install torch==1.13.0+cu117 torchvision==0.14.0+cu117 torchaudio==0.13.0 --extra-index-url https://download.pytorch.org/whl/cu117

# Install clip and other dependencies
pip install ftfy regex tqdm
pip install git+https://github.com/openai/CLIP.git
pip install scikit-image pandas einops thop

# If your NumPy version is 2.x, you need to uninstall it and install version 1.x
pip uninstall numpy
pip install numpy==1.23.5
```

#### Step1. Download and place weights.
All pre-trained weights can be downloaded: 
<a href="https://pan.baidu.com/s/1rBY8HN1T0_n1Z5rXo0KIHw?pwd=kaq9">Baidu Disk (pwd:kaq9)</a>] \[<a href="">Google Drive (TBD)</a>]

Then place the files into the correct directories:
- `E2P_model.tar` → `ckpts`
- `model_iter_12000_klrate.pth` and  `prompt_iter_12000_klrate.pth` → `prompt-ckpt`
- `ViT-B-32.pt` → `clip_model`


#### Step2. Place your degraded images.
Put your degraded image to `input` directory.
#### Step3. Run and get clear image！
You can run this project with the command `python test-E2P.py`.
The restored clear images will be stored in the `output` folder.
